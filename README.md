# QueryMind AI

QueryMind AI is an LLM-powered SQL Data Analyst Assistant that converts natural language business questions into safe SQL queries, executes them on PostgreSQL, and returns results with explanations and visualizations.

## Features

- Natural language to SQL generation
- PostgreSQL query execution
- SQL safety validation
- Interactive full-stack dashboard
- Result tables and charts
- Query history
- Dockerized development setup

## Tech Stack

- React + TypeScript + Vite
- FastAPI
- PostgreSQL
- SQLAlchemy
- OpenAI API
- Docker
